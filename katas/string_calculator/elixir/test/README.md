# StringCalculator kata

Main goal: take a string and add the numbers from it.
Secondary goal: 
* Handle and returns string separator errors
* Handle numbers error and return an error message

## Processing

1. Separate the numbers.
2. Parse numbers from string into numbers.
3. Add them.
4. Turn back the result into a string

## Errors handle for each part

1. Two separator in a row as in '3.2,\n2'.
2. Parsing error.
3. Not supposed to happen if we handled error before
4. Not supposed to happend if we handled error before
